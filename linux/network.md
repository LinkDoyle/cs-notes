# ufw - Uncomplicated Firewall

```bash
sudo ufw enable
sudo ufw disable

sudo ufw status
sudo ufw status verbose
sudo ufw status numbered

sudo ufw allow 22
sudo ufw insert 1 allow 80

sudo ufw deny 22
sudo ufw delete deny 22

sudo ufw --dry-run allow http

```
